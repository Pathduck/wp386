wp386
=====

![Screenshot](screenshot.png?raw=true)

A [live demo](http://themes.kkob.us/wp386/) is available.

wp386 is a WordPress theme based on the Bootstrap theme [BOOTSTRA.386](https://github.com/kristopolous/BOOTSTRA.386).
